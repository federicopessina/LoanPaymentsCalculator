# LoanPaymentsCalculator

Program that calculates the future expiration for a loan in case of monthly payment.

## Format
Console application

## Input
From three input:
1. capital
2. interest rate
3. span of the dept in months

## Output
Calculate, for each deadline

1. payments due
2. instalments

## Design Pattern
MVC
